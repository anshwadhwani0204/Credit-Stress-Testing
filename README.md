# Credit Stress Testing

## Overview

This project performs credit stress testing on a loan portfolio by analyzing rating migrations and estimating expected credit losses over a 10-year horizon.

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Rating transition matrices
- Probability of Default (PD) estimation
- Expected Loss (EL) calculation
- Portfolio risk analysis
- Visualizations using Matplotlib and Seaborn

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The notebook imports the dataset directly from Google Sheets using `gdown`.

## Results

The project estimates yearly transition probabilities, portfolio PD, and expected losses under credit stress scenarios.
